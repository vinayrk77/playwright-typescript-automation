# Playwright TypeScript Automation

End-to-end test automation framework built with 
Playwright and TypeScript.

## Tech Stack
- Playwright
- TypeScript  
- Node.js

## What's Being Built
- UI automation with Page Object Model
- API testing with Playwright
- CI/CD with GitHub Actions
- Cross-browser testing

## Status
🚀 Active development - adding tests daily

## Run Tests
npm install
npx playwright test

## Test Report
npx playwright show-report
